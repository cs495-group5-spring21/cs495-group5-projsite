Senior Design Group 5:

ARIA Website Revamp

Reworking the website and archive of the Alabama Research Institute on Aging

The team at ARIA has asked us to update and improve their outdated UA website. Their current framework consists of two sites, both based in Wordpress: a public-facing informational site and a private archive with tools used by ARIA researchers. The client wants both these sites to be combined with improved user security, better database search structuring, and updated graphic design.

Project Goals

Create New ARIA Site

using Bootstrap templates provided by UA's Division of Strategic Communications

Utilizing UA's collection of page templates, we will be able to conform to the design standards required of a University organization while easily porting over existing information from the old ARIA site.

Move and Restructure Archive

Firebase

Moving away from Wordpress, we will port all pdf documents from the old archive to a cloud firestore and provide better searching functionality.

Admin Analytics

with User Accounts

Create a comprehensive admin dashboard that displays analytics from users who access the archive. The dashboard will also allow admins to edit core page information, like faculty and blog posts.

News Feature

Add web blogging feature accessible in admin dashboard to populate homepage with current news, lecture videos, and events from the team at ARIA.
